# Memory-augmented Message Passing(MeMP)
![MeMP](./img/full_workflow.drawio.svg)

## Install 
```bash
conda create -n memp python=3.11.10
conda activate memp
export CXX=g++
pip install torch torch-geometric torch-sparse torch-lightning ogb
pip install scikit-learn numba tqdm colorama uniplot PyYAML yacs sphinx
```
